## 📝 記録について (About Logs)

日々の練習内容や課題感、気づきは [`log.csv`](file:///d:/code/Guitarlog/log.csv) に蓄積しています。

### 直近の課題・取り組み
* **運指の正確性**: 弦を押さえる際、隣の弦（例：6弦を押さえる際に5弦）に触れてしまいミュートされてしまう課題の克服。
* **ピッキングの安定性**: ルート弾きで音が途切れず、滑らかに（ﾍﾞﾍﾞﾍﾞﾍﾞﾍﾞと）鳴らせるようにする基礎練習。
