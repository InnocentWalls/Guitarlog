## 📝 記録について (About Logs)

日々の練習内容や課題感、気づきは [`log.csv`](file:///d:/code/Guitarlog/log.csv) に蓄積しています。また、演奏目標とする楽曲リストと各チューニングの設定は [`songs.md`](file:///d:/code/Guitarlog/songs.md) にまとめています。

### 直近の課題・取り組み

* **運指の正確性とフォーム（改善中）**: 
  * 人差し指が隣の弦に触れてしまう課題は大幅に減少！
  * 代わりに、指を立てたことで手のひらが6弦に触れてしまう新たな課題が発生。ネック裏の親指の位置を下げ、手のひらとネックの間に空間を作るフォームを練習中。
* **リフの運指とポジション移動**: 
  * `Satisfaction` のリフで4・5フレットへの指のストレッチに挑戦中。無理に指を開かず、手全体をスライドさせるポジション移動でスムーズに弾く方法を実践。
* **ピッキングの安定性と音作り**: 
  * ピッキングの音が途切れないよう、手首の脱力やピックのアングル調整を意識。
  * BOSS GX-10のファズ（FUZZ）などのエフェクトを活用し、狙った音色で気持ちよく弾く感覚を養う。
